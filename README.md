## ğŸ›  Getting Started

### Prerequisites

- Node.js (>=16)  
- npm or yarn  

### Installation

```bash
git clone https://github.com/thungan1909/easyEnglish.git
cd easyEnglish
npm install   # or yarn install
```

### Development

```bash
npm run dev   # or yarn dev
```

Runs at [http://localhost:5173](http://localhost:5173) (default Vite port).
